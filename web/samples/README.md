# Sample clips

Two 15-second excerpts bundled so visitors can try the demo without supplying their own audio file — one lyrical/melodic, one rhythmically driven, since that contrast is what the CIS vs. FS4 strategies are actually about.

Both are trimmed from recordings by pianist **Paul Pitman**, produced by [Musopen](https://musopen.org) (a nonprofit that commissions and dedicates classical recordings to the public domain) and mirrored on Wikimedia Commons.
